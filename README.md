# Discord RPC Client

Discord RPC client for Rust

> *Note*: Only works on **Unix** systems right now.

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
discord-rpc-client = "^0.1"
```

## Examples

Examples can be found in the examples directory in this repo.
